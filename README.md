# AI_ASSIGNMENT_2_INDIVIDUAL_PROGRAMMING

Assignment 1 – Simple Turing Test

This program simulates a very basic version of the Turing Test.
The program asks the user a few questions and evaluates the responses based on things like response time, length of answers, and language patterns.

At the end, it makes a rough guess about whether the responses look human-like or bot-like.
It’s a simplified demonstration meant to show the idea behind how machines might try to distinguish human behavior from automated responses.

Assignment 2 – Simple CAPTCHA

This program implements a basic CAPTCHA system.
The user is given a small challenge (such as a simple math problem) that must be solved before continuing.
